개인정보처리방침

최종 수정일: 2026년 2월 18일

Link Summarizer AI(이하 '본 서비스')는 사용자의 개인정보를 소중히 여기며, 관련 법령을 준수하기 위해 최선을 다하고 있습니다. 본 방침은 서비스 이용 과정에서 수집되는 정보와 처리 방법을 설명합니다.

1. 수집하는 정보의 항목

본 서비스는 원활한 요약 기능을 제공하기 위해 다음과 같은 정보를 일시적으로 처리할 수 있습니다:

  • URL 데이터: 사용자가 요약을 요청한 웹 페이지의 주소
  • 웹 페이지 콘텐츠: 요약을 위해 추출된 해당 웹 페이지의 텍스트 데이터
  • API 설정 정보: 사용자가 직접 입력한 Gemini 또는 Groq API 키 (로컬 스토리지에 저장됨)

2. 개인정보의 수집 및 이용 목적

수집된 정보는 오직 다음의 목적으로만 이용됩니다:

  • 사용자가 요청한 웹 페이지 내용의 분석 및 요약 생성
  • 서비스 품질 개선 및 오류 진단

3. 제3자 서비스 제공

본 서비스는 요약 기능을 수행하기 위해 사용자가 선택한 AI 모델 제공사로 데이터를 전송합니다:

데이터 전송처: Google Gemini API 또는 Groq API

전송 항목: 웹 페이지 텍스트 콘텐츠

전송된 데이터는 각 공급업체의 개인정보처리방침에 따라 처리됩니다.

4. 개인정보의 보유 및 파기

본 서비스는 서버에 사용자의 데이터를 저장하지 않습니다. 모든 요약 요청은 API 호출을 통해 즉시 처리되며, 처리 완료 후 전송된 데이터는 본 서비스 측에서 보유하지 않습니다.

API 키와 설정 정보는 사용자의 브라우저 내 '로컬 스토리지'에만 저장되며, 사용자가 확장 프로그램을 삭제하거나 데이터를 직접 삭제할 때까지 유지됩니다.

5. 사용자의 권리

사용자는 언제든지 확장 프로그램 설정 페이지에서 API 키를 수정하거나 삭제할 수 있으며, 확장 프로그램을 브라우저에서 삭제함으로써 모든 서비스 이용 및 데이터 처리를 중단할 수 있습니다.
