# 🚀 AI Link Summarizer (Chrome Extension)

AI를 활용하여 링크 위에 마우스를 올렸을 때 해당 페이지의 핵심 내용을 2~3문장으로 요약해주는 크롬 익스텐션입니다. **Gemini 1.5 Flash** 모델을 사용하여 빠르고 정확한 요약을 무료로 제공합니다.

## ✨ 주요 기능
- **지능형 호버 감지**: 의도치 않은 마우스 이동에는 반응하지 않고, 링크에 머물 때만 작동합니다.
- **Glassmorphism UI**: 세련된 반투명 블러 효과가 적용된 팝업 창을 제공합니다.
- **자동 한국어 요약**: 해외 사이트 링크라도 AI가 자동으로 한국어로 번역 및 요약합니다.
- **무료 AI 사용**: 사용자가 자신의 Gemini API Key를 입력하여 무료로 서비스를 이용할 수 있습니다.

## 🛠 주요 업데이트 및 문제 해결 (Troubleshooting)

개발 과정에서 발생한 주요 문제점들과 이를 해결하기 위해 적용된 최적화 기술입니다.

### 1. API 연결 안정성 확보 (Model Auto-Detection)
- **문제**: 일부 환경이나 API 키에 따라 `gemini-1.5-flash` 모델을 찾지 못하는 "Model not found" 에러가 발생했습니다.
- **해결**: `gemini-flash-latest`, `gemini-1.5-flash`, `gemini-pro` 등 사용 가능한 여러 모델 별칭과 API 버전(v1, v1beta)을 순차적으로 시도하는 **Fallback(자동 검색)** 로직을 구현하여 모든 사용자 환경에서 최적의 모델이 연결되도록 수정했습니다.

### 2. UI/UX 반응성 최적화 (2-Step Hover Logic)
- **문제**: 마우스를 드래그하거나 빠르게 이동할 때 의도치 않게 요약 창이 뜨는 불편함이 있었고, 반대로 요약 결과를 기다리는 체감 시간이 길게 느껴졌습니다.
- **해결**:
    - **Step 1 (250ms)**: 마우스가 링크에 진입하고 0.25초가 지나면 백그라운드에서 즉시 API 호출을 시작합니다 (UI는 아직 보이지 않음).
    - **Step 2 (800ms)**: 0.8초가 지났을 때만 실제 요약 창을 띄웁니다.
    - **결과**: 사용자가 요약창을 보게 되는 시점에는 이미 데이터가 준비되어 있어 **'즉각적으로 결과가 나오는'** 듯한 쾌적한 속도감을 구현했습니다.

### 3. Manifest V3 호환성 (DOM Error Fix)
- **문제**: 서비스 워커(background.js)에서 웹 페이지를 파싱할 때 `DOMParser`를 사용할 수 없어 에러가 발생했습니다.
- **해결**: 정규표현식(Regex) 기반의 경량 텍스트 추출 로직으로 대체하여 V3 표준에 맞는 안정성을 확보했습니다.

## 🚀 설치 및 사용법

1. **파일 다운로드**: 이 프로젝트 폴더를 로컬 PC에 저장합니다.
2. **익스텐션 로드**:
   - 크롬 브라우저에서 `chrome://extensions` 접속.
   - 오른쪽 상단 **'개발자 모드'** 활성화.
   - **'압축해제된 확장 프로그램을 로드합니다'** 클릭 후 폴더 선택.
3. **API 키 설정**:
   - 확장 프로그램 옵션(설정) 페이지에서 [Google AI Studio](https://aistudio.google.com/)의 API 키를 입력하고 **Save**를 누릅니다.
4. **테스트**: 뉴스 사이트(네이버, CNN 등)에 접속하여 링크 제목 위에 마우스를 0.8초간 올려보세요!

## 📂 프로젝트 구조
- `manifest.json`: 익스텐션 설정 및 권한 정의
- `content.js`: 호버 감지 및 UI 노출 로직 (2단계 딜레이 적용)
- `background.js`: 페이지 컨텐츠 추출 및 Gemini API 연동 (자동 모델 탐색 포함)
- `styles.css`: Glassmorphism 디자인 시스템
- `options.html/js`: 사용자 API 키 관리용 세팅 페이지
