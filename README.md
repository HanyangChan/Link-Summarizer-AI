## 🛠 주요 업데이트 및 문제 해결 (Troubleshooting)

개발 과정에서 발생한 주요 문제점들과 이를 해결하기 위해 적용된 최적화 기술입니다.

### 1. API 연결 안정성 확보 (Model Auto-Detection)
- **문제**: 일부 환경이나 API 키에 따라 `gemini-1.5-flash` 모델을 찾지 못하는 "Model not found" 에러가 발생했습니다.
- **해결**: `gemini-flash-latest`, `gemini-1.5-flash`, `gemini-pro` 등 사용 가능한 여러 모델 별칭과 API 버전(v1, v1beta)을 순차적으로 시도하는 **Fallback(자동 검색)** 로직을 구현하여 모든 사용자 환경에서 최적의 모델이 연결되도록 수정했습니다.

### 2. UI/UX 반응성 및 안정성 최적화 (Robust 2-Step Hover Logic)
- **문제**: 마우스를 드래그하거나 빠르게 이동할 때 의도치 않게 요약 작업이 시작되는 불편함이 있었고, 링크 내부의 요소(아이콘, 텍스트 등) 위에서 커서를 움직이면 타이머가 초기화되는 문제가 있었습니다.
- **해결**:
    - **Step 1 (500ms)**: 마우스가 링크에 진입하고 0.5초가 지나면 백그라운드에서 요약 데이터 페칭을 시작합니다. (순간적인 이동에는 작동하지 않도록 딜레이 강화)
    - **Step 2 (1200ms)**: 1.2초가 지났을 때만 실제 요약 창을 띄웁니다.
    - **안정성 강화**: `relatedTarget` 감지를 통해 링크 내부 요소들 간의 이동 시에는 타이머가 초기화되지 않도록 개선하여 더욱 안정적인 호버 효과를 구현했습니다.
    - **결과**: 의도치 않은 작동(False Positive)을 획기적으로 줄이면서도, 요약창이 뜨는 시점에는 데이터가 준비되어 있어 즉각적인 결과 확인이 가능합니다.


### 3. Manifest V3 호환성 (DOM Error Fix)
- **문제**: 서비스 워커(background.js)에서 웹 페이지를 파싱할 때 `DOMParser`를 사용할 수 없어 에러가 발생했습니다.
- **해결**: 정규표현식(Regex) 기반의 경량 텍스트 추출 로직으로 대체하여 V3 표준에 맞는 안정성을 확보했습니다.

### 4. 확장 가능한 AI 구조 (Multi-Provider Support)
- **문제**: Gemini 무료 티어의 호출 횟수 제한(Rate Limit)으로 인해 사용량이 많을 때 서비스가 중단되는 문제가 있었습니다.
- **해결**: OpenAI 호환 API를 사용하는 **Groq** 연동 기능을 추가했습니다. 사용자는 옵션 페이지에서 더 빠른 속도와 넉넉한 할당량을 가진 엔진을 직접 선택할 수 있습니다.

### 5. 글로벌 다국어 지원 (i18n - KO, EN, JA)
- **문제**: 글로벌 사용자 대응을 위해 UI와 요약 결과물이 해당 국가의 언어로 제공되어야 할 필요가 있었습니다.
- **해결**: `chrome.i18n` API를 사용하여 **한국어, 영어, 일본어**를 정식 지원합니다. 현재 브라우저의 UI 언어를 감지하여 AI에게 해당 언어로 요약하도록 지시하는 동적 프롬프트 로직을 구현했습니다.

### 6. 스토어 자산 제작 및 최적화
- **업데이트**: 스토어 등록을 위한 아이콘 및 프로모션 타일을 제작했습니다. 특히 병아리 마스코트 테마를 추가하여 친근감을 높였으며, 규격 오류를 방지하기 위해 자동 리사이징 스크립트(`resize_assets.py`)를 개발하여 모든 자산을 `store_assets` 폴더에 최적화된 상태로 구성했습니다.

## 🚀 설치 및 사용법

1. **파일 다운로드**: 이 프로젝트 폴더를 로컬 PC에 저장합니다.
2. **익스텐션 로드**:
   - 크롬 브라우저에서 `chrome://extensions` 접속.
   - 오른쪽 상단 **'개발자 모드'** 활성화.
   - **'압축해제된 확장 프로그램을 로드합니다'** 클릭 후 폴더 선택.
3. **API 키 설정**:
   - 확장 프로그램 옵션(설정) 페이지에서 **AI Provider**를 선택합니다.
   - [Google AI Studio](https://aistudio.google.com/) 또는 [Groq Cloud](https://console.groq.com/keys)에서 API 키를 발급받아 입력하고 **Save**를 누릅니다.
4. **테스트**: 뉴스 사이트(네이버, CNN 등)에 접속하여 링크 제목 위에 마우스를 1.2초간 올려보세요! (Groq 선택 시 훨씬 빠른 응답을 체감할 수 있습니다.)

## 📂 프로젝트 구조
- `manifest.json`: 익스텐션 설정 및 권한 정의 (i18n 지원)
- `_locales/`: 다국어 메시지 리소스 (KO, EN, JA)
- `content.js`: 호버 감지 및 UI 노출 로직
- `background.js`: 페이지 컨텐츠 추출 및 AI API 연동 (동적 언어 요약 포함)
- `i18n-helper.js`: HTML 요소 자동 번역 유틸리티
- `styles.css`: Glassmorphism 디자인 시스템
- `options.html/js`: 사용자 설정 페이지 (다국어 지원)
- `store_assets/`: 최적화된 스토어용 그래픽 자산
